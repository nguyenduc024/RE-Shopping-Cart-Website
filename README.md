# 🛒 RE Shopping Cart Website

## 📌 Overview
This project is a **basic e-commerce clothing website** developed as part of my learning journey, based on a starter source code provided by **Rikkei Academy**.

Instead of simply replicating the original project, I focused on:
- Understanding the **full application flow**
- Refactoring and improving parts of the logic
- Applying additional technical skills to make the project more structured and maintainable

> 🎯 **Goal:** Build a solid foundation in frontend development and understand how a real-world web application operates.

---

## 🚀 Features
- Display clothing products
- Add / remove / update items in shopping cart
- Dynamic UI updates using JavaScript
- User interaction handling
- Basic UI structure designed for scalability
